

# 🎨 ENG240 Developers — Full Color System

## 🌿 Primary Brand Color (Identity)

Used for main buttons, logo accents, active states.

```css
Primary: #10B981
Dark: #059669
Light: #D1FAE5
```

---

## 🌊 Secondary Color 1 (Tech / Links / UI Glow)

Used for links, hover effects, info highlights.

```css

Cyan Dark: #0891B2
Cyan Light: #A5F3FC
```

---

## ⚡ Secondary Color 2 (Warnings / Highlights / Energy)

# 🧠 How to Use This System

## Buttons

* Primary → Green
* Secondary → Cyan
* Danger → Red
* Special → Violet

---

## Tags

* Success → Green
* Tech → Cyan
* Warning → Amber
* Featured → Violet

---

## Sections

* Background → Dark neutrals
* Cards → Slightly lighter dark
* Highlights → Cyan/Violet gradients
